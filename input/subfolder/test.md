# サブフォルダテスト

このファイルはサブフォルダに配置されたテストファイルです。

## 機能確認

- **recursive: true** の場合、このファイルも変換対象になります
- **recursive: false** の場合、このファイルは無視されます

## Mermaidテスト

```mermaid
graph TD
    A[サブフォルダのMD] --> B[PDF変換]
    B --> C[output/subfolder/test.pdf]
```

## 結果

フォルダ構造が保持されて `output/subfolder/test.pdf` に出力されます。
