Markdown PDF Converter (Portable)

USAGE:
1. Double-click convert.bat
2. Check output folder for PDFs

WHAT IT DOES:
- Converts Markdown files to PDF
- Supports Mermaid diagrams
- Japanese text support
- No installation required

REQUIREMENTS:
- Windows 64-bit
- 4GB RAM
- 500MB disk space

FILES:
- markdown-pdf-converter.exe (main program)
- convert.bat (launcher) 
- input/ (sample files)
- output/ (PDF output)
- config.json (settings)

CUSTOM USAGE:
markdown-pdf-converter.exe "input_folder" "output_folder"
markdown-pdf-converter.exe --single "file.md" "output.pdf"
